CORS Playground · ローカル用の検証UI

CORS Anywhereっぽいフロントエンド

これはブラウザから直接CORSを魔法で消すものではありません。そんな便利すぎる話はたいてい裏にプロキシが要ります。 この画面は、そのプロキシにリクエストを投げるためのテスト用UIです。

リクエスト作成

GET / POST / PUT / PATCH / DELETE
想定しているプロキシ形式は、/{target-url} に中継するタイプです。 例えば http://localhost:8080/https://api.example.com/data のような形です。 実際のバックエンドの仕様に合わせて調整してください。

結果

レスポンスをそのまま表示
待機中 未送信
送信先 -
最終ステータス - 
ここにレスポンスが出ます。人類がまた一つ、便利そうなUIを作った気分になれる場所です。
補足: このHTMLだけでは、別ドメインの制約を解除できません。CORS対策は通常、サーバー側で許可設定を行うか、同一オリジンのプロキシを使います。 これはその操作を試すためのフロントエンドです。
